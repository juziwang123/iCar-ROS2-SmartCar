# 灯光控制 - 快速启动指南

## 📌 核心功能

| 场景 | 灯光行为 | 说明 |
|------|---------|------|
| 开机 | 🟢 常亮 | 前后灯全亮，表示小车就绪 |
| 直行 | 🟢 常亮 | 保持开机状态 |
| **左转** | 🟡 左灯闪烁 | 左转信号灯，500ms 闪烁 |
| **右转** | 🟡 右灯闪烁 | 右转信号灯，500ms 闪烁 |
| **急停** | 🔴 全灯快闪 | 危险警报，200ms 快速闪烁 |

## ⚡ 一键启动

### 仿真环境

```bash
cd ~/ros2_ws
source install/setup.bash

# 方式1：启动完整系统（控制+灯光+雷达）
ros2 launch car_bringup bringup.launch.py

# 方式2：仅启动控制和灯光
ros2 launch car_control control.launch.py
```

### 实车硬件

```bash
# 需要修改配置文件启用 GPIO
# 编辑 src/car_control/config/control.yaml
# 将 enable_gpio_control: false 改为 enable_gpio_control: true

# 然后启动（需要 root 权限）
sudo ros2 launch car_control control.launch.py
```

## 🧪 测试灯光

### 方式1：自动化测试（推荐）

```bash
# 终端1：启动灯光控制
ros2 launch car_control control.launch.py

# 终端2：运行测试脚本（会自动演示所有模式）
python3 tests/test_light_control.py
```

预期输出：
```
[light_tester]: 灯光控制测试开始
[light_tester]: [测试1] 启动 - 灯光常亮
[light_controller]: Light mode: CONSTANT

[light_tester]: [测试3] 左转 - 左转指示灯闪烁 (500ms)
[light_controller]: Light mode: TURN_LEFT

[light_tester]: [测试6] 急停 - 所有灯快速闪烁 (200ms)
[light_controller]: Light mode: BLINK_FAST
```

### 方式2：手动命令测试

**终端1 — 启动灯光**
```bash
ros2 run car_control light_controller
```

**终端2 — 测试转向**

左转：
```bash
ros2 topic pub /control/cmd_vel geometry_msgs/Twist '{linear: {x: 0.2, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: 0.5}}'
```

右转：
```bash
ros2 topic pub /control/cmd_vel geometry_msgs/Twist '{linear: {x: 0.2, y: 0.0, z: 0.0}, angular: {x: 0.0, y: 0.0, z: -0.5}}'
```

急停：
```bash
ros2 topic pub /emergency_stop std_msgs/Bool '{data: true}'
```

### 方式3：实时监听灯光模式

```bash
ros2 topic echo /car/light_control
```

输出：
```
data: 1  # CONSTANT (常亮)
---
data: 4  # TURN_LEFT (左转)
---
data: 5  # TURN_RIGHT (右转)
---
data: 3  # BLINK_FAST (急停)
```

## 🔧 关键参数配置

编辑 `src/car_control/config/control.yaml`：

```yaml
/**:
  ros__parameters:
    # 灯光参数
    enable_gpio_control: false      # 仿真:false, 实车:true
    turn_threshold: 0.3             # 转向判断阈值 (rad/s)
    blink_slow_interval: 0.5        # 转向灯闪烁间隔 (秒)
    blink_fast_interval: 0.2        # 急停闪烁间隔 (秒)
    
    # GPIO 映射（仅实车有效）
    front_left_gpio: 17
    front_right_gpio: 27
    rear_left_gpio: 22
    rear_right_gpio: 23
    brake_gpio: 24
```

## 🐛 故障排查

| 问题 | 原因 | 解决方案 |
|------|------|--------|
| 仿真中看不到灯光效果 | 这是正常的 | 查看日志或监听 `/car/light_control` 话题 |
| 实车灯不亮 | GPIO 编号不对或权限不足 | 检查配置并用 `sudo` 运行 |
| 灯光不随转向变化 | `/control/cmd_vel` 速度过小 | 增大转向角速度阈值或检查速度命令 |
| 启动时报 GPIO 错误 | GPIO 库未安装 | 实车需要 `RPi.GPIO` 库 |
