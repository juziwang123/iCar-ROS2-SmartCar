# 灯光控制模块使用说明

## 功能概述

本模块实现了 iCar 智能小车的自动灯光控制系统，支持：

- ✅ **开机常亮** — 小车启动后前后灯常亮
- ✅ **转向闪烁** — 左转/右转时对应方向灯闪烁（500ms 间隔）
- ✅ **急停报警** — 触发急停时所有灯快速闪烁（200ms 间隔）
- ✅ **双控制模式** — 支持 ROS2 话题控制（仿真）和 GPIO 直接控制（实车）

## 架构设计

### 控制链路

```
灯光控制器
    ↓
订阅: /control/cmd_vel (速度命令)
订阅: /emergency_stop (急停信号)
    ↓
发布: /car/light_control (灯光模式)
    ↓
┌─────────────────────────────┐
│  GPIO 模式（实车硬件）       │ ← 控制真实 LED
│  Topic 模式（仿真/调试）     │ ← 发布灯光模式消息
└─────────────────────────────┘
```

### 灯光模式定义

| 模式 ID | 模式名称 | 状态 | 用途 |
|--------|---------|------|------|
| 0 | OFF | 全灭 | 不使用 |
| 1 | CONSTANT | 常亮 | 正常行驶、停车 |
| 2 | BLINK_SLOW | 慢闪 | 预留（500ms） |
| 3 | BLINK_FAST | 快闪 | 急停报警（200ms） |
| 4 | TURN_LEFT | 左转闪烁 | 左转时左转灯闪烁 |
| 5 | TURN_RIGHT | 右转闪烁 | 右转时右转灯闪烁 |

### 灯光分组

- **前左灯** — GPIO 17 (前转向灯)
- **前右灯** — GPIO 27 (前转向灯)
- **后左灯** — GPIO 22 (后转向灯)
- **后右灯** — GPIO 23 (后转向灯)
- **制动灯** — GPIO 24 (刹车灯，可选)

## 启动方法

### 方法1：作为 control.launch.py 的一部分启动（推荐）

```bash
cd ~/ros2_ws
source install/setup.bash

# 启动整个控制模块（包括灯光控制）
ros2 launch car_control control.launch.py
```

### 方法2：单独启动灯光控制节点

```bash
# 启动灯光控制节点（需要先启动 ROS2 master）
ros2 run car_control light_controller
```

### 方法3：在仿真中启动

```bash
# 启动 bringup（包含所有控制节点）
ros2 launch car_bringup bringup.launch.py use_control:=true use_lights:=true
```

## 配置说明

### 仿真环境配置（推荐用于测试）

编辑 `src/car_control/config/control.yaml`：

```yaml
/**:
  ros__parameters:
    # ... 其他参数 ...
    
    # 灯光控制参数
    light_control_topic: /car/light_control
    enable_gpio_control: false  # ← 仿真时设为 false
    front_left_gpio: 17
    front_right_gpio: 27
    rear_left_gpio: 22
    rear_right_gpio: 23
    brake_gpio: 24
    turn_threshold: 0.3       # 左右转判断阈值 (rad/s)
    blink_slow_interval: 0.5  # 慢闪间隔 (秒)
    blink_fast_interval: 0.2  # 急停快闪间隔 (秒)
```

### 实车硬件配置

在小车上启动时，编辑 `control.yaml`：

```yaml
    enable_gpio_control: true   # ← 实车时设为 true
    front_left_gpio: 17         # ← 根据实际 GPIO 布线调整
    front_right_gpio: 27
    rear_left_gpio: 22
    rear_right_gpio: 23
    brake_gpio: 24
```

**注意**：GPIO 编号需根据实际硬件布线配置。如果小车已有预定义的 GPIO 映射，应该咨询设备文档或之前的项目记录。

## 工作逻辑

```python
if 急停信号:
    所有灯 → 快速闪烁 (200ms)
elif 转向角速度 > 阈值:
    if 左转 (angular.z > 0):
        左转灯 → 闪烁
    else:
        右转灯 → 闪烁
else:
    所有灯 → 常亮
```

## ROS2 话题接口

### 订阅话题

| 话题 | 消息类型 | 说明 |
|------|---------|------|
| `/control/cmd_vel` | geometry_msgs/Twist | 车辆速度命令，用于判断转向 |
| `/emergency_stop` | std_msgs/Bool | 急停信号，优先级最高 |

### 发布话题

| 话题 | 消息类型 | 说明 |
|------|---------|------|
| `/car/light_control` | std_msgs/Int32 | 当前灯光模式 (0-5) |

## 参数说明

| 参数 | 类型 | 默认值 | 说明 |
|------|------|--------|------|
| `enable_gpio_control` | bool | false | 是否启用 GPIO 硬件控制（仿真:false, 实车:true） |
| `turn_threshold` | float | 0.3 | 转向判断阈值 (rad/s)，超过此值判定为转向 |
| `blink_slow_interval` | float | 0.5 | 慢闪间隔 (秒) |
| `blink_fast_interval` | float | 0.2 | 急停快闪间隔 (秒) |
| `front_left_gpio` | int | 17 | 前左灯 GPIO 编号 |
| `front_right_gpio` | int | 27 | 前右灯 GPIO 编号 |
| `rear_left_gpio` | int | 22 | 后左灯 GPIO 编号 |
| `rear_right_gpio` | int | 23 | 后右灯 GPIO 编号 |
| `brake_gpio` | int | 24 | 制动灯 GPIO 编号（可选） |

## 常见问题

### Q1: 仿真环境中看不到灯光变化？

**A**: 这是正常的。仿真环境不支持 GPIO 硬件控制。可以通过以下方式验证：

1. 监听 `/car/light_control` 话题查看模式变化
2. 查看 ROS2 节点日志输出（`Light mode: ...`）
3. 运行自动化测试脚本查看模式转换

### Q2: 实车上灯不亮？

**A**: 检查以下几点：

1. **GPIO 编号** — 验证配置文件中的 GPIO 编号与实际硬件布线一致
   ```bash
   # 查看 GPIO 状态
   cat /sys/class/gpio/gpio17/value
   ```

2. **权限问题** — GPIO 访问需要 root 权限
   ```bash
   # 使用 sudo 运行
   sudo ros2 run car_control light_controller
   ```

3. **GPIO 初始化失败** — 查看日志
   ```bash
   ros2 run car_control light_controller | grep -i gpio
   ```

4. **硬件连接** — 确认 LED 电源、地线、GPIO 引脚都正确连接

### Q3: 如何自定义转向灯颜色或闪烁模式？

**A**: 修改 `light_controller.py` 中的 `_set_directional_lights()` 和 `_schedule_blink_toggle()` 方法。例如：

```python
# 修改转向灯闪烁间隔
def _start_blink_directional(self, direction: str) -> None:
    # ... 
    self._schedule_blink_toggle(0.3)  # 改为 300ms
```

### Q4: 如何禁用灯光控制？

**A**: 有两种方法：

1. **不启动灯光节点**
   ```bash
   ros2 launch car_control control.launch.py use_lights:=false
   ```

2. **在 launch 文件中注释掉灯光节点**
   编辑 `launch/control.launch.py`，注释 light_controller 节点
